Materialize CSS
  • Used by: Amazon, Nasa, Unity, etc.
  • Browser compatibility
  • Responsive 12-grid system utilizing columns, gutters, margins
  • Aesthetically-pleasing
  • React-based components
  • Can be customizable
  • Smaller community support

Bootstrap
  • Used by: Twitter, Apple Music, Airbnb, etc.
  • Browser compatibility
  • Responsive 12-grid system utilizing containers, rows, columns
  • Responsivity and functionality
  • Can be a complex framework
  • Consistent and user-friendly
  • Larger community support
